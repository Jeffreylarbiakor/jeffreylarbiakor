## 🏆 Featured project

### 🏦 [Credit Default Prediction — Home Credit Default Risk](https://github.com/Jeffreylarbiakor/credit-default-prediction-home-credit)

End-to-end credit scoring model on 307,511 loan applications. LightGBM tuned via Optuna, stratified 5-fold CV, SHAP interpretation, business-aware threshold analysis. Built and shipped in two days; full methodology and limitations documented in the repo.

| Metric | Value |
|---|---|
| ROC-AUC (CV) | **0.7692** |
| PR-AUC (CV) | **0.2536** |
| KS statistic | **0.402** *(acceptable→strong boundary)* |
| Top-decile precision | **27.77%** *(3.4× population base rate)* |
| Defaulters captured in top decile | **34.4%** |

**Standout finding:** the two engineered features (`EXT_SOURCE_MEAN`, `CREDIT_TERM`) occupy 2 of the top 3 SHAP positions — each outperforming the raw features they were derived from.
